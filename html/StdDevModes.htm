Standard Deviation Modes

Location: Program Preferences Window

The F12 key cycles through the four standard deviation modes:

For H2O with molecular weight 18.01528 and std. dev. of 0.0003159114,

  • Short mode displays the result as 18.0153(3)
  • Scientific mode displays it as 18.0153 (3.159114E-04)
  • Decimal mode displays it as 18.01528 (.0003), std. dev. not rounded
  • Off mode displays it as 18.01528, without standard deviation or rounding.

The standard deviation (accuracy) of calculated molecular weights and percent compositions is calculated according to mathematical formulas.
