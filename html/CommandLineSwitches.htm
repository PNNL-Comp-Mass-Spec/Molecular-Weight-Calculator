Command Line Switches

Syntax: MWTWIN [/F:filename] [/O:outfile] [/Y] [/X] [/Z] [/?]

Available options are:

/F:filename to start MWTWIN in batch mode, calculating weights for compounds in filename and writing results to filename.out

/O:outfile to specify the name of the output file for batch mode

/Y to overwrite outfile without prompting the user

/X to prevent the program from attempting to write abbreviation, element, or default value files to disk

/Z to prevent the program from attempting to read or write abbreviation, element, or value files

/? or /Help to display this screen.

See MWTWIN.CHM for more help.
