Program Availability

Windows 9x/NT/ME/00/XP Version (MWTWin v6.x)

The program is available via the web at http://www.alchemistmatt.com/ and http://ncrr.pnl.gov/software/
The installation zip file (for example mwt6_39.zip) contains MSVBVM60.DLL and a full installation program, utilizing the Windows MSI Installer.

Windows 3.x Version (MWTWin v3.2)

The program is available via the web at http://www.alchemistmatt.com/ and http://www.geocities.com/alchemistmatt/ in the mwt3_2.zip file.

Dos Version (MWT v2.87)

The program is available via the web at http://www.alchemistmatt.com/ and http://www.geocities.com/alchemistmatt/ in the mwt2_87.zip file.
