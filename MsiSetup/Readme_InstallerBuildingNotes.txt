Note: If the MwtWindll.dll file is changed, then you must update the MwtwinDll Merge Module (MwtWindll.msm file) 
using ..\MwtWinDll\MwtWinDll_MergeModule\MwtWinDll_MergeModule.sln then copy that file to 
C:\Program Files (x86)\Microsoft Visual Studio\Common\Tools\VSInst\BuildRes prior to re-building the MwtWin.exe installer

Similarly, after recompiling CWSpectrumDll.dll, open CWSpectrum_MergeModule.sln 
to re-create CWSpectrum_MergeModule.msm and copy it to 
C:\Program Files (x86)\Microsoft Visual Studio\Common\Tools\VSInst\BuildRes
